Get Involved NJ App
Python

This is the final project I made for my python class at Kean University. The goal was to create an app that aggregated all of the important information from local governments so that it was easy for people to get involved. I started by scraping the local government websites and displaying it in a GUI. While working on the project I found out that the information I was expecting to find was not available or was hard to find and parse. If I were to do it over again, I would focus on creating a guide that explained more about how local governments work so that the work would reach more people. Still, I am happy with the progress I made and learning about front end with python, local governments and web scraping.
